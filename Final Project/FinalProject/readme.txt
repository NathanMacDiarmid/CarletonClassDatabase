Instructions for database:

- navigate to where FinalProject.zip was downloaded and unzip files
- navigate to filepath where the files were unzipped through terminal
- use command sqlite3 FinalProject.db, this will open the database
- when you are finished looking at the database itself, type .exit
- to run the UI for the database, in the same directory as the unzipped files type python FinalProject.py
-> ALT - you can also open the FinalProject.py in your favourite code editor and run the program
-> ALT - this will also allow you to see the sql queries performed for the program
- close the database when you are done
